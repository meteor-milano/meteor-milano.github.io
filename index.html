Meteor Milano

Meteor Milano è il primo meetup milanese su Meteor, il javascript framework open-source.

Perché scegliere Meteor

Meteor è uno dei framework javascript emergenti. Fornisce una soluzione completa per lo sviluppo e permette di ottenere un sito reattivo scrivendo il codice una volta sola.

Primo MEETUP

Vogliamo condividere le nostre conoscenze su Meteor, farvi vedere qualche applicazione pratica e qualche segreto per sfruttarlo al meglio delle sue potenzialità.

Il MeetUp sarà a Milano in via Fabio Filzi 5, nella sede di Dev4side.
Data: metà gennaio. Stay tuned!
